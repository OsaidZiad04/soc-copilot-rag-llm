SOC Copilot
Analyze security files in a clear workflow: upload, chunk, index, then ask directly in chat.
This project is built for SOC teams so one uploaded report or log becomes stored content, split into chunks, indexed for retrieval, and then used consistently across analysis and chat.
Begin from File Analysis, then move to AI Chat and ask about the same indexed file.
How It Works
1
Upload the file
Upload PDF, TXT, or LOG from File Analysis inside the same project.
2
Let the system process it
The file is stored, split into chunks, and indexed automatically for retrieval.
3
Analyze and ask
You get IOCs, rules, and summary, then chat answers from that same file content.
